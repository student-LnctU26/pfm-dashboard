# PFM Dashboard

A Personal Finance Management Dashboard built using the MERN stack.

## Features

- User Authentication
- Expense Tracking
- Budget Management
- Charts & Analytics

## Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js

## Project Structure

client/ -> frontend  
server/ -> backend

## Run Backend

cd server  
npm install  
npm run dev

## Run Frontend

cd client  
npm install  
npm run dev
